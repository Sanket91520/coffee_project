Overview
This project analyzes 149,116 coffee vending machine transactions to identify sales trends, peak hours, product performance, and store-level insights.
The dashboard was built using Power BI, DAX, and Time-Series Forecasting.

Dataset
Transaction-level data including:
Date & Time
Quantity & Unit Price
Store Location
Product Category & Type

Objectives
Analyze daily & monthly sales trends
Identify peak sales hours
Evaluate product performance
Compare store locations
Forecast next 7 days of sales

Tools Used
Power BI
DAX
Power Query
Time-Series Forecasting (95% Confidence)

Key Insights
Sales peak during morning & evening hours
Weekends generate higher revenue
Certain product categories dominate total sales
Clear time-based patterns support short-term forecasting

Data Cleaning
